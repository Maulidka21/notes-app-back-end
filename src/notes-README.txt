4 Berkas(file) yg dibutuhkan untuk project aplikasi catatan

server.js: 
Memuat kode untuk membuat, mengonfigurasi,
dan menjalankan HTTP server menggunakan Hapi.

routes.js:
Memuat kode konfigurasi routing server, seperti
menentukan path, method, dan handler yang digunakan.

handler.js: 
Memuat seluruh fungsi-fungsi handler
yang digunakan pada berkas routes.

notes.js:
Memuat data notes yang disimpan dalam bentuk array objek.
